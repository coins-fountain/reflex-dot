# Reflex Dot

Reflex Dot is a fast-paced, addictive mobile game where your speed and reflexes are put to the ultimate test.

**Goal:** Tap the dot before it shrinks and vanishes!

## Features

- **Dynamic Gameplay:** The game gets harder as your score increases, with the dot shrinking faster and becoming more challenging to catch.
- **High Score Tracking:** Compete against yourself to beat your personal best.
- **Revive System:** Watch a rewarded ad to continue your streak after a game over.
- **Ad Integration:**
  - **Banner Ads:** Displayed unobtrusively on the Idle and Game Over screens.
  - **Interstitial Ads:** Shown periodically (every 3rd game over) to monetize the app efficiently.
  - **Rewarded Ads:** Optional ads that give you a second chance.
- **Responsive Design:** Optimized for both Portrait and Landscape orientations.

## Getting Started

This project is built with [Flutter](https://flutter.dev/).

### Prerequisites

- Flutter SDK (latest stable version recommended)
- Android Studio / Xcode for mobile development

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/reflex_dot.git
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the app:
    ```bash
    flutter run
    ```

## Technology Stack

- **Flutter** & **Dart**
- **GetX** for State Management
- **Google Mobile Ads** for Monetization
- **Get Storage** for Local Data Persistence
- **Package Info Plus** for Version Display
